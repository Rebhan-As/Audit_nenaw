Aksi Diperlukan: Aturan Keamanan Salah

Database Anda menolak koneksi. Ini **bukan kesalahan aplikasi**, melainkan pengaturan keamanan Anda.

Langkah-langkah Perbaikan:

  1. Buka Firebase Console dan pilih proyek: .
  2. Navigasi ke Build > Firestore Database > Rules.
  3. Hapus semua teks yang ada dan ganti dengan aturan berikut:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth.uid == userId; } } }
  1. Klik tombol biru "Publish".
  2. Tunggu 2 menit penuh, lalu refresh halaman ini.
