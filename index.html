Aksi Diperlukan: Perbaiki Izin Akses Database

Aplikasi ini diblokir oleh database Firebase Anda. Untuk memperbaikinya, Anda **harus** memperbarui Aturan Keamanan (Security Rules) di Firebase Console.

Langkah-langkah Perbaikan:

  1. Buka Firebase Console dan pilih proyek: .
  2. Navigasi ke Build > Firestore Database > Rules.
  3. Hapus semua teks yang ada di editor aturan.
  4. Ganti dengan aturan berikut yang mengizinkan akses hanya untuk pengguna yang sudah login:
rules_version = '2'; service cloud.firestore { match /databases/{database}/documents { match /users/{userId}/{document=**} { allow read, write: if request.auth.uid == userId; } } }
  1. Klik tombol biru "Publish".
  2. Tunggu 1-2 menit, lalu refresh halaman ini.
